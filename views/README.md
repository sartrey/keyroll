# keyroll

`keyroll` is a secure key-value storage toolkit.

## Usage

### install as global dependency

`npm install -g keyroll`

### start server anywhere

```sh
# serve ~/.keyroll at 9988
keyroll

# serve ~/.keyroll at 8080
keyroll --port=8080

# fork server
keyroll fork

# stop server
keyroll stop

# show help
keyroll help
```

### access homepage

`http://localhost:YOUR-PORT`
